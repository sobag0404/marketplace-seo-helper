# 🏷️ Marketplace SEO Helper

**Бесплатный SEO-генератор хештегов для продавцов на Ozon, Wildberries и других маркетплейсах.**

Загрузите .xlsx файл с товарами → получите релевантные хештеги → скачайте результат. Всё работает прямо в браузере — никакие данные не уходят на сервер.

## ✨ Возможности

- 📊 **Загрузка Excel/CSV** — автоматическое определение столбцов (название, артикул)
- 🏷️ **Генерация хештегов** — на основе 16 предустановленных категорий товаров
- 🇷🇺 **Хештеги на русском** — приоритет русскоязычных тегов, английский только для брендов
- 🔍 **Поиск по категориям** — быстрый поиск нужной категории товаров
- ⚙️ **Настройка минимума** — укажите минимальное количество хештегов (по умолчанию 10)
- ✏️ **Редактирование** — добавляйте, удаляйте и редактируйте хештеги вручную
- 📋 **Массовое копирование** — копируйте хештеги для всех товаров одним кликом
- 📤 **Экспорт** — скачивайте результат в Excel или CSV формате
- 📈 **Аналитика** — частотный облако хештегов, оценки качества, статистика генерации
- 🎯 **Пакетные операции** — применяйте действия к выбранным строкам
- 🔄 **Отмена действий** — Ctrl+Z для возврата к предыдущему состоянию
- 🎨 **Тёмная/светлая тема** — переключение одним кликом

## 🚀 Деплой на Vercel

1. Форкните или клонируйте этот репозиторий
2. Перейдите на [vercel.com](https://vercel.com)
3. Нажмите **"Add New Project"**
4. Выберите ваш GitHub-репозиторий
5. Vercel автоматически определит Next.js и настроит сборку
6. Нажмите **"Deploy"**

Всё! Ваш сайт будет доступен по адресу `your-project.vercel.app`

## 🛠️ Локальная разработка

```bash
# Установка зависимостей
bun install

# Запуск dev-сервера
bun run dev

# Сборка для продакшена
bun run build

# Запуск продакшен-сборки
bun run start
```

## 🏗️ Технологии

- **Next.js 16** — React-фреймворк
- **TypeScript** — типобезопасность
- **Tailwind CSS 4** — стилизация
- **shadcn/ui** — UI-компоненты
- **SheetJS (xlsx)** — обработка Excel/CSV файлов
- **Framer Motion** — анимации
- **Lucide Icons** — иконки

## 🔒 Приватность

Все данные обрабатываются **исключительно в вашем браузере**. Файлы не загружаются на сервер, хештеги генерируются локально с помощью словарей и правил.
